# LiteSignin 每日签到插件

LiteSignin 是一个 Minecraft Bukkit/Spigot 服务器插件，允许玩家通过在线游玩自动签到并获得奖励。

## 功能特性

- **自动签到**: 玩家在线超过设定时间后自动签到
- **可配置奖励**: 支持物品奖励和经济货币奖励
- **经济插件集成**: 支持 Vault 插件，兼容多种经济系统
- **灵活配置**: 所有参数均可在配置文件中自定义
- **数据持久化**: 玩家签到数据自动保存
- **图形界面**: 提供直观的日历式签到界面
- **随机奖励**: 支持固定数量和随机范围的奖励
- **签到限制**: 每个玩家每天只能签到一次
- **奖励开关**: 可独立控制物品奖励和经济奖励的启用状态

## 安装说明

1. 将编译好的 `.jar` 文件放入服务器的 `plugins` 文件夹
2. 启动服务器，插件会自动生成配置文件
3. 根据需要修改 `config.yml` 配置文件
4. 重启服务器或使用 `/reload` 命令重新加载插件

## 配置说明

### 基本设置
```yaml
settings:
  # 是否启用签到消息提示
  enable_signin_message: true
  
  # 签到成功消息
  signin_message: "§a[LiteSignin] §f签到成功！奖励已发放。"
  
  # 是否启用GUI界面
  enable_gui: true
```

### 奖励设置
```yaml
rewards:
  # 物品奖励
  items:
    # 是否启用物品奖励
    enabled: true
    
    # 物品列表
    list:
      # 示例: 给予钻石 (固定数量)
      - material: DIAMOND
        amount: 1
        
      # 示例: 给予金锭 (随机数量范围)
      - material: GOLD_INGOT
        amount_range:
          min: 1
          max: 10
  
  # 经济奖励
  money:
    # 是否启用经济奖励
    enabled: true
    
    # 固定金额示例
    amount: 100.0
    
    # 随机金额范围示例 (注释掉固定金额后启用此项)
    #amount_range:
    #  min: 50.0
    #  max: 200.0
    
    # 货币名称
    currency: "金币"
```

### 数据库设置
```yaml
database:
  # 存储类型 (sqlite, mysql)
  type: "sqlite"
  
  # SQLite 设置
  sqlite:
    path: "signin_data.db"
```

## 命令

- `/signin` - 文本签到命令
- `/signin gui` - 打开签到GUI界面
- `/sigui` - 打开签到GUI界面 (别名: `/sgui`, `/signgui`)

## 权限

- `litesignin.use` - 使用签到功能 (默认: true)
- `litesignin.admin` - 管理员权限 (默认: op)

## GUI界面特性

- **日历视图**: 按月份显示完整的日历界面
- **状态指示**: 不同颜色的羊毛表示不同签到状态
  - 🟨 黄色: 今天(未签到)
  - 🟩 绿色: 已签到
  - 🟥 红色: 过去未签到
  - ⬜ 灰色: 未来日期
- **月份导航**: 支持前后月份切换
- **防重复点击**: 内置冷却机制防止刷屏

## 经济插件支持

本插件通过 Vault API 支持以下经济插件：
- EssentialsX Economy
- CraftConomy
- MultiCurrency
- 其他兼容 Vault 的经济插件

## 编译

使用 Maven 编译插件：
```bash
mvn clean package
```

## 依赖

- Spigot/Paper API 1.21+
- Vault API (可选，用于经济奖励)
